## When to Use

Use this workflow when creating or evaluating a cover for any KDP book. Run the thumbnail test and 8-mistake checklist before every publish. Use the tier system to match your budget to the right production path.

## The Framework

### The Thumbnail Test (Do This First)

"90% of readers will see your book as a tiny thumbnail first. If your text is unreadable or the design fades at small scale, you're finished. You're cooked. It's over." -- Dale L Roberts

1. Shrink your cover to roughly 120 x 180 pixels (Amazon search result size)
2. Can you read the title? If not, increase font size or boost contrast
3. Does the design go muddy or fade? Simplify elements, increase contrast
4. Does it signal the correct genre at a glance?
5. Does it have a visual hook that stops scrolling?

### Eight Cover Mistakes That Kill Sales

| # | Mistake | Fix |
|---|---------|-----|
| 1 | Using KDP's Cover Creator | Use Canva (free) or hire a designer |
| 2 | Bad typography | Serif = timeless; Bold = importance; Handwritten = elegance; Grunge = mystery |
| 3 | Wrong genre signals | Study top 20 books in your Amazon category and match visual language |
| 4 | Stock photos and generic templates | Professional design blends imagery, typography, and composition |
| 5 | Unreadable at thumbnail size | Bold typography, strong contrast, test at 120x180 |
| 6 | Weak composition | Balanced layout, complementary colors, polished type |
| 7 | Over-complicated concept | Focus on ONE strong idea |
| 8 | No visual hook | Unusual color, single powerful symbol, or character in tension |

"A cover is more than artwork. It's a promise. When readers see your book for the first time, they're not just looking at colors and shapes. They're measuring your professionalism, your genre fit, and whether your story feels worth their time." -- Dale L Roberts

### Tier 1: DIY with Canva ($0)

"This is a great way to get started with Amazon KDP if you're low in budget because you can basically create a book cover for free. Now once you start making money from your books, I highly recommend you reinvest a profit into hiring an actual illustrator." -- Sean Dollwet

**Ebook front cover:**
1. Canva > Templates > search "[genre] book cover" > filter FREE
2. Modify the template heavily -- delete watermarks, reposition elements
3. Add a symbolic graphic matching the book's theme
4. Title text in genre-appropriate font, subtitle with keywords
5. Download as PNG

"If it's a template on Canva, chances are high that a lot of other people are also using it as well." -- Sean Dollwet

**Paperback wrap cover:**
1. KDP Cover Template Generator > enter trim size + page count > download template
2. Canva > Create Custom Size (switch units to inches) > enter template dimensions
3. Upload KDP template as guide layer
4. Position front cover PNG on the RIGHT side
5. Design back cover with matching elements
6. Download as PDF Print with CMYK color profile (requires Canva Pro)

**Export settings:**

| Format | File Type | Color Profile |
|---|---|---|
| Ebook cover | PNG | RGB |
| Paperback wrap | PDF Print | CMYK (Canva Pro) |
| Hardcover wrap | PDF Print | CMYK (Canva Pro) |

### Tier 2: Budget Designers on Fiverr ($5-$30)

Search for designers specializing in book covers within your genre. Order from sellers with 4.5+ ratings and 50+ completed orders. Provide: genre, comp titles (3-5 top-selling covers in your category), title, subtitle, author name, and any symbolic elements.

### Tier 3: Professional Designers ($100-$500+)

Platforms: Reedsy (curated marketplace), 99designs (competition-based), Miblart (book-specific). Dave Chesson on Reedsy: the marketplace curates designers and editors and facilitates communication with milestones.

"You only pay when you're happy." -- Miblart

### Series Branding Strategy

When publishing multiple books in a series, maintain consistent visual elements across all covers: same font family, same color palette, same layout structure, same author name placement. Readers recognize series branding at thumbnail size and are more likely to buy subsequent titles.

## Example

Budget: $0 for first book, reinvest profits into Tier 2 for books 2-5.

Book 1 (self-help, $0): Canva template modified with custom graphics, Georgia Pro font, thumbnail-tested at 120x180px. Passes all 8 mistake checks.

Books 2-5 ($20 each on Fiverr): Same genre visual language, consistent series branding, professional typography. Each cover thumbnail-tested before upload.

## Output

A publication-ready cover package includes:
- Ebook front cover (PNG, RGB, minimum 2,560 x 1,600 pixels)
- Paperback full wrap (PDF Print, CMYK, sized to exact page count)
- Hardcover full wrap if applicable (PDF Print, CMYK, different dimensions)
- Thumbnail test passed at 120x180 pixels
- 8-mistake checklist cleared
- Series branding elements documented for future titles
