## When to Use

Use this workflow after your manuscript is written and your cover is designed. The two-phase framework applies whether you are formatting in Word, Kindle Create, Atticus, or Vellum. Use the tool comparison to choose the right formatter for your situation, and the print QA checklist before every paperback upload.

## The Framework

### The Two Phases of Book Formatting

Dave Chesson draws a line most self-publishers miss: "Designing your manuscript and formatting that into a book. The process and settings you use between these two different phases is wildly different."

**Phase 1 -- Manuscript Setup (while writing):**
- Standard page size (8.5 x 11)
- Times New Roman 12pt
- Heading 1 on every chapter title, Heading 2 on subtitles
- Page breaks between chapters (Insert > Page Break -- never hit Enter repeatedly)
- Scene breaks: three centered asterisks (* * *)
- Headers: last name / title on left, page number on right

**Phase 2 -- Book Formatting (after writing):**
- Change page size to trim size
- Set mirror margins
- Justify text
- Custom line spacing
- Page numbers and headers
- Fix widows and orphans
- Ensure even page count
- Export as PDF (print) or ePub (ebook)

Everything in Phase 1 feeds Phase 2. Skip heading styles or use tabs for indentation during writing, and you pay for it with hours of cleanup.

### Phase 2 Settings (Microsoft Word -- Print)

**1. Trim size.** Common sizes:

| Trim Size | Typical Use |
|---|---|
| 5 x 8" | Fiction paperback (most popular) |
| 6 x 9" | Non-fiction, self-help |
| 8.5 x 11" | Children's books, coloring books |

**2. Mirror margins.** "When you open up a physical book notice that the size of the outside margin is different from the size of the inside margin." -- Dave Chesson. Starting point for 150-300 pages at 5x8: inside 0.375", outside 0.35".

**3. Line spacing.** Select all text. Line spacing > Exactly > 15pt. Paragraph spacing before/after to zero. Target ~30 lines per page.

**4. Fix indentation.** Special > First Line > 0.3 inches. "The most common mistake we see authors make is just hitting the tab button before a paragraph or hitting the spacebar multiple times." -- Dave Chesson

**5. Page numbers.** Bottom, center. Uncheck "show on first page."

**6. Headers.** Book title on odd pages, author name on even pages.

**7. Widows and orphans.** Disable automatic handling. Manually adjust line height on problem pages.

**8. Even page count.** "You need to ensure you've ended with an even number of pages because this is required by most printing companies." -- Dave Chesson

**9. Export as PDF** for print upload.

### Sean's Quick eBook Method (Under 10 Minutes)

1. Turn on Show/Hide marks
2. Delete extra paragraph marks, replace with page breaks
3. Apply Heading 1 to every chapter title
4. Justify all body text
5. Build Table of Contents: References > Custom > uncheck page numbers > check hyperlinks
6. Clean up TOC entries
7. Add back-matter hyperlinks ("Click here to leave a review")

### Tool Comparison

| Tool | Cost | Platform | Best For | Limitation |
|---|---|---|---|---|
| Microsoft Word | Already owned | PC/Mac | Full control, both phases | Steep learning curve for print |
| Kindle Create | Free | PC/Mac | Amazon-only, simple books | KPF output (Amazon only), no footnotes, larger file size |
| Atticus | $147 one-time | PC/Mac/iPad/Chromebook | Cross-platform, ePub + PDF | Newer product |
| Vellum | $249 one-time | Mac only | Polished output, loyal users | Mac-only, expensive |

On Kindle Create's hidden cost: "Amazon will charge you the author 15 cents per megabyte for your file size every time somebody buys and downloads your ebook... atticus.io creates file sizes 2.5 times less than that of Kindle Create." -- Dave Chesson

### File Format Reference

| Destination | File Format | Notes |
|---|---|---|
| KDP ebook | ePub (preferred) or DOCX | ePub honors formatting best |
| KDP print | PDF | Never upload DOCX for paperback |
| Other retailers | ePub | Universal standard since MOBI deprecation |

### Print QA Checklist

- [ ] Trim size matches KDP template
- [ ] Mirror margins set for page count
- [ ] Text justified, line spacing consistent
- [ ] First-line indentation via paragraph settings (not tabs)
- [ ] Page numbers present, hidden on title/copyright pages
- [ ] Chapter titles all use Heading 1
- [ ] Even total page count
- [ ] Widows/orphans manually fixed
- [ ] Cover dimensions match interior page count (spine width)
- [ ] Exported as PDF

## Example

A 200-page self-help book at 6x9 trim:
- Phase 1: Written in Word with Heading 1 chapters, page breaks, Times New Roman 12pt
- Phase 2: Changed to 6x9, mirror margins (inside 0.4", outside 0.35"), 15pt line spacing, first-line indent 0.3"
- Ebook: Exported as ePub via Atticus (drag-and-drop import from Word)
- Print: Exported as PDF from Word, cover wrap generated from KDP template calculator

## Output

Publication-ready files:
- ePub file for ebook upload (or KPF if using Kindle Create)
- PDF file for print interior upload
- Print QA checklist completed
- File size noted (for delivery fee calculation on ebook)
